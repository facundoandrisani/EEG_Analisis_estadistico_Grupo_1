# Descriptores Estadísticos del EEG
Este trabajo se enfoca en caracterizar el comportamiento temporal de las señales EEG. El objetivo principal es segmentar la actividad cerebral antes, durante y después de una crisis epiléptica para analizar cómo varían métricas como la varianza, la covarianza y la autocorrelación.

Además, busca establecer umbrales de detección automatizada y encontrar qué función de densidad de probabilidad (PDF) modela mejor la naturaleza de estos datos.
